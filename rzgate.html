RZGate: Instant HTTP Access to Roomzin

A lightweight, high-performance HTTP/JSON proxy that lets any system tap into Roomzin's sub-millisecond inventory engine — without native SDK integration.

Many teams love Roomzin's speed but work with existing HTTP-based platforms or legacy booking engines that aren't ready for a native SDK yet. RZGate is our lightweight HTTP/JSON proxy that makes adoption instant and risk-free.

Immediate access

Key Features

HTTPS + HTTP/2 + CORS

Production-ready TLS, full CORS, and HTTP/2 for maximum performance.

Token-Based RBAC

Full-access and read-only tokens — hot-reloadable without restart.

Prometheus Metrics

GET /metrics — full observability.

Minimal & Scalable

Single lightweight binary — easy horizontal scaling.

HTTP/2 is fully supported and strongly recommended
it delivers significantly higher throughput and lower latency than HTTP/1.1.
